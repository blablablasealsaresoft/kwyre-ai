# Installer

Platform-specific installers for all Kwyre products.

- `install_windows.ps1` — Windows CLI installer (firewall rules, deps, service)
- `install_windows_gui.py` — Windows GUI installer (tkinter wizard, 5-page flow)
- `install_linux.sh` — Linux installer (systemd + iptables)
- `install_macos.sh` — macOS installer (launchd + PF firewall)
