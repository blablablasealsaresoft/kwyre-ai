# Chat / Frontend

Static HTML frontend served by all Kwyre backends.

- `main.html` — Main product page with live chat demo, floating orbs, news ticker
- `technology.html` — Competitor incidents feed + benchmark comparison
- `custom.html` — Custom LLM/SLM service page with industry orbs
- `security.html` — Tiger Team security page, privacy guarantees, compliance
- `platform.html` — Deployment options and platform support
- `pay.html` — Product selection + payment (XMR/CC)
- `landing.html` — Cinematic intro page
- `index.html` — Splash/entry page
